Evidence and Impact - Module Guide

This page contains the slide content for the Newspeak House Evidence and Impact module (2025-26).

Module Slides (2025-26)

  • Evaluation & Impact in Civic Technology | Session Summary (Oct 14, 2025)
    Introduction to evaluation approaches in civic tech, covering causality, mixed methods, and designing meaningful metrics. Includes case studies from mySociety, CitizenLab, and Sure Start.

  • Statistics 101 for Evaluation | Session Summary (Nov 2, 2025) Foundational statistics for rigorous evaluation: understanding distributions, sample sizes and variance; using p-values and confidence intervals in experiments; planning with power calculations; interpreting regressions; and understanding Bayesian approaches to inference.

Previous Year’s Module: Public Sector Innovation (2024-25)

  • Delivering Innovation in Public Institutions
    Introduction to the module covering implementation challenges, delivery obstacles, and how to navigate pushback when implementing innovation in public sector organizations.

  • Evaluation Masterclass
    Deep dive into rigorous evaluation methods including RCTs, quasi-experimental designs, and treatment effects. Features case studies from Read Easy UK, Code Club, Skills for Life, and D.A.R.E.
